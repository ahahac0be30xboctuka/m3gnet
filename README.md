|---|---|---|
| **А** | Прогнал инференс на 5 равновесных кубических кристаллах (Si, Cu, Ni, LiF, Mo) | Силы ~$10^{-7}$ эВ/Å (численный ноль), инференс < 2 мс/структуру |
| **Б** | Релаксация Mo BCC с растянутой решёткой $a=3.3$ Å → 3.168 Å (MP-эталон 3.16762 Å) | 39 шагов FIRE за ~4 с, отклонение от MP +0.008%; supercell $2{\times}2{\times}2$ с возмущениями 0.1 Å - ~350 шагов, BCC-симметрия восстановилась |

| **Г** | **Гипотеза H2:** «Пре-релаксация снижает ошибку энергии минимум в 10× по сравнению с прямым предсказанием». 5 металлов с решёткой $\pm 5\%$ от равновесия | **Подтвердилась с запасом.** Прямое предсказание ошибается на 137 мэВ/атом в среднем, после релаксации: 0.09 мэВ/атом. ## Структура репо

```
.
├── README.md           
└── scripts/
    ├── README.md       - инструкция по установке и запуску
    ├── expA_predict.py + expA_results.json
    ├── expB_relax.py   + expB_results.json
    ├── expC_classes.py + expC_classes_results.json
    └── expD_prerelax.py + expD_prerelax_results.json
```

Сырые результаты лежат рядом со скриптами в JSON. Чекпоинт модели в репо **не включён** - `m3gnet` качает его сам при первом вызове (или можно указать локальный путь через переменную окружения `M3GNET_CHECKPOINT`).

## Окружение

macOS arm64 (Apple Silicon), conda Python 3.10, `tensorflow-macos==2.13.0`, `m3gnet==0.2.4`, `pymatgen==2024.2.20`, `ase==3.22.1`. Команды установки находятся в [`scripts/README.md`](scripts/README.md).
